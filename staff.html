How to Use

1
To register a new towel — scan or type Tag ID, select type, press Register
2
To dispatch — scan Tag ID, select location, press Dispatch to Room
3
To return — scan Tag ID, press Return to Laundry
4
To clean — scan Tag ID after washing, press Mark as Clean
5
To retire — scan Tag ID, press Retire when towel reaches end of wash life (200 cycles)
6
To delete — type Tag ID manually (can't scan defective tag), press Delete. Use only for broken RFID chips

USB Reader Tip

Keep this page open and the Tag ID field focused. When you wave a tag near the USB reader it will automatically type the ID and press Enter — no clicking needed.
